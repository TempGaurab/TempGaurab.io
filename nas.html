North American Steels Data Analysis


This link to this project is not shown as the proejct involved data of North American Steels which is highly confidential. We did the following analysis for the project:

  1. Merged diverse Steel datasets into a unified structure using Pandas
  2. Implemented data cleaning tasks, including deduplication, anomaly detection, and correction using Pandas and Numpy
  3. Addressed missing data and ensured consistency.
  4. Developed a modularized Python application for efficient data processing.
  5. Uploaded cleaned data to the designated database using sqlite3.
  6. Analyzed the datasets to provide meaningful outcomes.
